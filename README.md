# 🚗 AutoTwinAI: AI-Powered Automotive Supply Chain Intelligence

AutoTwinAI is an AI-powered supply chain intelligence platform designed to help automotive manufacturers proactively identify supply chain risks, optimize procurement decisions, and improve operational resilience through data-driven insights.

The platform combines supplier intelligence, commodity analytics, predictive machine learning, and interactive dashboards to provide a centralized control tower for supply chain monitoring and decision-making.

---

## 📌 Features

* 📊 **Supply Chain Control Tower** for real-time monitoring
* 🌍 **Supplier Risk Scoring** using multiple risk indicators
* 📈 **Commodity Intelligence Dashboard**
* 🔄 **Alternate Supplier Recommendations**
* 🤖 **Machine Learning-based Risk Prediction**
* 📦 **ERP & Inventory Monitoring**
* 📉 **Production Risk Simulation**
* 📍 Interactive dashboards built with Streamlit

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/punitd-source/AutoTwinAI.git
```

Move into the project directory

```bash
cd AutoTwinAI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app/dashboard.py
```

## 📈 Workflow

1. Collect supplier, commodity, and manufacturing data.
2. Preprocess and merge datasets.
3. Generate supplier risk scores.
4. Train machine learning models.
5. Forecast potential supply chain risks.
6. Visualize insights through an interactive Streamlit dashboard.
